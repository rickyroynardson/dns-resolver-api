## Description

Express.ts api to check domain dns presence and validity

## Installation

```bash
$ npm install
```

## Running the app locally

Copy .env.example file into .env and fill the variable

```bash
# migrate prisma schema
$ npx prisma migrate dev

# start development
$ npm run start:dev
```

## Build with

- Express
- Prisma
- PostgreSQL
- Node Cron
- Winston (Logger)
